# Lessons

Project corrections and durable workflow lessons go here.

## Current Rules

- Keep detailed project documentation in `docs/`.
- Keep `AGENTS.md` at the repo root for Codex.
- Use a design-system-first UI approach.
- Target WCAG 2.2 AA from the first scaffold.
- Preserve multi-resume lanes.
- Reuse CareerOps code and functionality as much as possible; do not reinvent
  scanner, PDF, tracker, evaluation, or application-assistant behavior without a
  documented reason.
- Work one phase at a time. After each phase, verify, pause for user testing,
  and commit only after explicit user approval.
