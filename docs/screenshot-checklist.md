# Screenshot Checklist

Before taking screenshots:

- Use fictional profile data
- Use fictional resume content
- Use fictional companies or public demo jobs
- Hide API keys
- Hide email addresses and phone numbers
- Clear browser bookmarks if visible
- Use consistent browser width
- Use light mode unless dark mode is part of the product story
- Avoid screenshots that show localhost errors or debug states

## Required screenshots

1. Dashboard onboarding wizard
2. Normal dashboard with action queue
3. Jobs table with filters, including the Preference column
4. Job detail overview
5. AI evaluation running or completed
6. Resume generation tab
7. Applications Kanban board
8. Interview Prep STAR story bank
9. Settings AI Providers tab
10. Settings Job Sources tab
